This is a script to parse the CSV file located in the resource file and post it to Glady's api.  To run this yourself:

- Download the script
- Create an .env file and include:
  1. GLADLY_POST_ITEMS_URL (Glady's post endpoint for creating items)
  2. GLADLY_USER_NAME (Email)
  3. GLADLY_API_KEY (User's API Key)
- run "npm start"
